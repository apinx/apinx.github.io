This page hosts different applications / games created and managed by apinx.dk.

App: Java Manifest Viewer
If you are tired of trying to figure out the import/exports in Java manifest files(for OSGI for example), this will help.
App: Web Automation Portal
Scheduled recurring Web Tasks using a self-created DSL. This can be used to track online prices, bitcoin address balance, car rental prices or whatever is required. It also has a powerful event engine to handle otherwise common tasks, like finding updates, sending emails and so forth.
App: Web Notifications
Building on top of scrapey.net, this portal allows common users to get notified on website changes, such as booking availability, stock value updates or whatever the heart desires without requiring any coding abilities.
